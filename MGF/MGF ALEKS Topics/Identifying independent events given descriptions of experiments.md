# Identifying independent events given descriptions of experiments

Independent events are two events that do not affect the probability of each other.  
If I replace the time back in, I am not changing the probability. (Independent)
If I remove the item, I have changed the probability. (Dependent)
If the 1st event HAS to happen in order for the 2nd event to happen then the 2nd dependent upon the 1st. (Dependent)
